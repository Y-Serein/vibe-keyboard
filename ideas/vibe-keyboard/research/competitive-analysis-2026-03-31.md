# 竞品分析报告 — 2026-03-31

## 核心结论
**"AI Agent 物理控制器" 品类完全空白**，没有任何量产硬件专门解决 AI Agent 会话切换、权限审批、多 Agent 监控。

## 直接竞品
- **Jesse Vincent 的 Vibecoding 键盘**: DIY 项目，Staples Easy Button + RPi Pico，非卖品
- **Rabbit Project Cyberdeck**: ~$500，7寸屏+40%键盘+Linux系统，定位完整笔记本而非控制器，未上市

## 主要间接竞品

### Elgato Stream Deck（最大威胁）
- Mini $60-80 / MK.2 $150 / Plus $160-200 / XL $250
- 已有 AI 插件（tomAIto），但无 Agent 状态显示、Allow/Deny、多会话切换
- 全球市场 2024 年 $4.3 亿，预计 2033 年 $11.8 亿（CAGR 13.2%）

### 其他
- Razer Stream Controller: $150-270，Loupedeck 软件平台
- Logitech MX Creative Console: $200，面向 Adobe 用户
- Monogram: $399-799，模块化，面向创意专业人士
- QMK/VIA 宏键盘: $10-80，无屏幕，需编程知识
