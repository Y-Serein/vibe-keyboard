# 硬件发射案例研究 — 2026-03-31

## Flipper Zero (最相关)
- KS 筹 $4.9M，38K 支持者（目标 $60K 的 81 倍）
- 核心策略: 全开源 + 社区先行 + YouTube 病毒传播
- 2023年收入 ~$80M+，累计销量超百万
- **关键**: 吉祥物策略 + 争议营销 + 几乎零付费广告

## Elgato Stream Deck (品类定义者)
- 2017 发布，创造了"Stream Deck"品类
- 从直播主 → 远程办公 → 开发者 逐步扩展
- 核心: SDK 开放 + 插件生态 + KOL 合作

## Teenage Engineering (设计驱动)
- OP-1 $1,999 依然秒售罄
- 策略: 极致工业设计 + 限量稀缺 + 零传统广告
- Pocket Operators $59-89 作为品牌入口产品

## 所有案例共同规律
**社区先行 + 内容驱动 + 产品自己会说话**
演示视频是跨越"为什么需要专用硬件"认知鸿沟最有效的手段。
